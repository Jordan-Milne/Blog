Unpacking the "Subplot"

Jordan Milne | Sun 08 December 2019

Having Troubles with Subplots?"

matplotlib might not be the quickest/easiest way to spin up visuals in python but it is extremely customizable. A powerful tool for plotting when using matplotlib is subplots. Subplots allow you to have multiple plots on one figure, but setting up these plots can get confusing. This blog post will outline how to correctly add key features to your subplot.

The first step is to import the necessary packages and data sets

import pandas as pd
import matplotlib.pyplot as plt
import numpy as np

df = pd.read_csv("drug-use-by-age.csv")
df2 = pd.read_csv("diamonds.csv")

Let's start with a plot that has 2 rows and 3 columns. The first parameters in .subplot() is rows and the second parameter is columns. Figure size can also be adjusted in .subplot(). Creating a title for all the plots or a super title as matplotlib would call it, as well as adjusting the area of the subplots can be down as shown below.

fig, ax = plt.subplots(2,3,figsize=(9,5))

fig.suptitle('How to Arrange Subplots')

plt.subplots_adjust(left=None, bottom=None, right=None, top=None, wspace=None, hspace=None)

Now that there are some empty plots, we can fill them with data. In the first line of code below we create the subplot and have an axes label of "ax" but it can be anything (like "axs" or "axes") as long as you are consistent when you call on it later. To access a certain plot in the subplot we can locate it by using ax[row,column]. So to fill the middle top plot with data, ax[0,1] is used so the plot knows we are targeting that subplot. Setting a title for a specific sublot is a little different than normal matplotlib plotting, again you start with identifying which sublot you want to give a title to with ax[0,0] and then add set_title() as seen below

fig, ax = plt.subplots(nrows=2, ncols=3, sharex=False, sharey=True,figsize=(9,5))

ax[0,1].scatter(df2['carat'],df2['price'])
ax[0, 0].set_title('Axis [0,0]')
#instead of plt use ax(x)

plt.show()

For 1D subplots only one parameter is needed in ax[] as shown below. In the subplots below it is shown how do change the visual details of the sublot

  • c changes the colour of the data
  • alpha changes the opacity of the data
  • marker changes the shape of each data point
  • linestyle well... changes the line style of the data
fig, ax = plt.subplots(nrows=1, ncols=2, sharex=False, sharey=False,figsize=(7,4))
x = np.linspace(0, 10, 25)

ax[0].scatter(df2['carat'],df2['price'], c='red', alpha = 0.01)
ax[0].set_title('Axis [0]')
ax[1].set_title('Axis [1]')
ax[1].plot(x, np.sin(x), c='green', marker = '+', linestyle='dashed')
fig.suptitle('Two Plots')
plt.show()

To give the sublots some flavour, use plt.style.use('seaborn') which gives the sublots a background and a grid. In the figure below the x axis is the same for both sublots so in plt.sublots(), sharex can be set to True. In addition, by adding label to the subplot a legend can then be made by adding ax[X].legend() (with X being the indicated subplot row or column as shown below)

plt.style.use('seaborn')
fig, ax = plt.subplots(nrows=2, ncols=1, sharex=True, sharey=False,figsize=(17,7))
x = np.linspace(0, 10, 25)

fig.suptitle('Sharing x per column, y per row')

ax[0].plot(df['age'],df['alcohol-use'], c='red', label = 'Alcohol')
ax[1].plot(df['age'],df['marijuana-use'], c='orange', label = 'Marijuana')
ax[0].legend()
ax[1].legend()
fig.suptitle('Drug Usage')
plt.show()
