How to Make Fireworks in Python

I used Turtle, from the python standard library to make art that resembles fireworks. I created a series of multiple concentric circles with random initial and final radius as well as random widths for each circle.

first step was to import turtle which is basically a on screen pen that you can program. After that I imported randint and numpy as I wanted the circles to be different from eachother like fireworks.

The function circles creates multiple of concetric circles that will pop in a random position on the screen as seen using circle_position. Then a colour for the circles is randomly selected from the set list colours. In the for loop the first circle diameter, last circle diameter and number of circles in between is randomly determined. The rest of the loop are turtle commands that draw a circle.

The next function make_circles simply loops the circle function to make a series of multiple concentric circles that look like fireworks.

import turtle
from random import randint
import numpy as np
turtle.speed(0) #turns off draw mode so circles pop up instantly
turtle.bgcolor('black') #background black


def circles():
    circle_position = (randint(-400,400), randint(-300,300))  #
    colours = ('blue', 'Deep Sky Blue', 'Chartreuse', 'Deep Pink', 'Aquamarine',
               'red', 'Yellow', 'white', 'purple', 'green', 'orange', 'grey')
    circle_colour = np.random.choice(colours)
    for i in range(randint(1,50), randint(100,200), randint(10,30)):
        turtle.width(randint(1,9))
        turtle.pencolor(circle_colour)
        turtle.penup()  #pen up
        turtle.setpos(circle_position) #setting random starting position
        turtle.right(90)    # Face South
        turtle.forward(i)   # Move one radius
        turtle.right(270)   # Back to start heading
        turtle.pendown()    # Put the pen back down
        turtle.circle(i)    # Draw a circle
        turtle.penup()      # Pen up while we go home
        turtle.home()       # Head back to the start pos

def make_circles(number):
    for i in range(number+1):
        circles()
make_circles(20)

turtle.exitonclick()
