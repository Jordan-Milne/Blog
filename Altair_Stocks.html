Making Interactive Visualizations in Python

Jordan Milne | Wed 15 April 2020

In this blog post I will be using the following two DataFrames (Apple's stock and the NASDAQ index) to show you how to make a simple plot in Matplotlib and then an interactive version of that plot in Altair.

Lately I have been analyzing stocks using python and the yfinance API. I quickly became frustrated with Matplotlib because I was used to interactive plots on analyst websties. I decided to take a dive into some next generation python plotting software to enhance my projects visualizations.

One key differences between Matplotlib and Altair is how the data is read. With Matplotlib you can give a column from any DataFrame and also perform mathematical operations upon it as seen in the example below where the "Y" value given to Matplotlib was df_apple['Close']/df_apple['Close'].iloc[0]-1.

Matplotlib

fig, ax = plt.subplots(1,figsize=(10,5))
ax.plot(df_apple['date'],df_apple['Close']/df_apple['Close'].iloc[0]-1, label='AAPL')
ax.plot(df_nas['date'],df_nas['Close']/df_nas['Close'].iloc[0]-1, label='NASDAQ')
ax.set(xlabel='Date', title='AAPL Compared to NASDAQ (^IXIC)')
ax.yaxis.set_major_formatter(FuncFormatter(lambda y, _: '{:.0%}'.format(y))) 
ax.grid()
plt.legend();

Altair

As for Altair, the way data is passed into Altair is less flexible then matplotlib. This can be seen as a blessing and a curse because it forces you NOT to perform DataFrame operations in the plotting code as seen in the example above. Below I show the necessary steps to structure the data so Altair can easily read it.

df_apple['stock'] = 'AAPL'
df_nas['stock']='NASDAQ'
df_apple['percent'] = round((df_apple['Close']/df_apple['Close'].iloc[0]-1)*100,2)
df_nas['percent'] = round((df_nas['Close']/df_nas['Close'].iloc[0]-1)*100,2)
df = pd.concat([df_apple[['percent','stock','date']],df_nas[['percent','stock','date']]])

In Altair, you create a Chart instance and give it the data (DataFrame). Next, we indicate what kind of Mark to use in the visualization. For a line plot, we want the 'line' mark, but there are many other types of plots avialble in Altair. Next we encode which is where we indicate which columns we want. Since we told Altair the DataFrame to pull from we can just pass it column names as a string and it will recognize them. Next we seperate the data and color by indicating to Altair to seperate color on the column 'stock'

Below is a simple example of how to get the same plot as the matplotlib chart above:

alt.Chart(df).mark_line(interpolate='basis').encode(
    x=alt.X('date', axis=alt.Axis(title='Date')),
    y=alt.Y('percent', axis=alt.Axis(title='Percent (%)')),
    color=alt.Color('stock', legend=alt.Legend(title="Stock"))
).properties(width=500, height=300,
title= 'AAPL Compared to NASDAQ (^IXIC)')

Now the below code is to make it interactive. I create a selection that recognizes the nearest point on the plot to your mouse called nearest. Next we make selectors which gets the x-value of the cursor using nearest for altair. The next step is more for aesthetic but we make a points which will produce point markers on the iteractive plot for the values. text generates the "Y" value of the positon the cursor on on the x-axis. rules is simply just creating a grey verticle line where the mouse is. Finally, the layer method puts all the peices of the chart together allowing us to "tooltip" with our mouse and read values. The last step is .interactive() which enables the chart axis to be dragged around and zoomed in and out using the mouse.

# Create a selection that chooses the nearest point & selects based on x-value
nearest = alt.selection(type='single', nearest=True, on='mouseover',
                        fields=['date'], empty='none')

# The basic line
line = alt.Chart(df).mark_line(interpolate='basis').encode(
    x=alt.X('date', axis=alt.Axis(title='Date')),
    y=alt.Y('percent', axis=alt.Axis(title='Percent (%)')),
    color=alt.Color('stock', legend=alt.Legend(title="Stock"))
).properties(
title='AAPL Compared to NASDAQ (^IXIC)')

# Transparent selectors across the chart. This is what tells us
# the x-value of the cursor
selectors = alt.Chart(df).mark_point().encode(
    x='date',
    opacity=alt.value(0),
).add_selection(
    nearest
)

# Draw points on the line, and highlight based on selection
points = line.mark_point().encode(
    opacity=alt.condition(nearest, alt.value(1), alt.value(0))
)

# Draw text labels near the points, and highlight based on selection
text = line.mark_text(align='left', dx=5, dy=-10).encode(
    text=alt.condition(nearest, 'percent', alt.value(' '))
)

# Draw a rule at the location of the selection
rules = alt.Chart(df).mark_rule(color='gray').encode(
    x='date',
).transform_filter(
    nearest
)

# Put the five layers into a chart and bind the data
alt.layer(
    line, selectors, points, rules, text
).properties(
    width=600, height=300
).interactive()

Please use your mouse to drag and zoom to explore the interactive plot below!
